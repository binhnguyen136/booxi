Comment utiliser booxi avec Lightspeed ?

Tutoriels    |   juillet 8, 2016

Ce tutoriel vous montre, étape par étape, comment paramétrer le module Lightspeed dans booxi, et comment créer une vente et un bon de travail dans Lightspeed à partir de vos rendez-vous dans booxi.

  • Comment activer le module Lightspeed.
  • Comment lier des services dans booxi avec des items de ventes de Lightspeed.
  • Comment lier un personnel dans booxi avec un employé dans Lightspeed.
  • Comment créer une vente dans Lightspeed depuis un rendez-vous booxi.
  • Comment créer un bon de travail dans Lightspeed à partir d’un rendez-vous booxi.
  • Comment importer d’un seul coup tous les clients de Lightspeed dans booxi.

Dans ce tutoriel, nous supposons que vous savez déjà comment utiliser Lightspeed et booxi séparément, que vous avez déjà complété votre profil d’entreprise booxi, et que vous avez ajouté les informations sur vos services et sur votre personnel. Si ce n’est pas le cas, commencez avec le guide suivant pour vous familiariser avec booxi : Tutoriel : Réviser son profil d’entreprise.

Vous devez d’abord vous connecter sur votre compte booxi puis suivre les étapes suivantes.

Étape 1 – Activez et paramétrez votre module Lightspeed

Afin d’utiliser booxi avec le point de vente Lightspeed, vous devez activer le module Lightspeed dans booxi, en utilisant vos identifiants Lightspeed (adresse courriel et mot de passe).

  1. Connectez vous sur votre compte booxi si ce n’est pas déjà fait.
  2. Dans le menu principal situé à gauche, cliquez sur l’onglet Mon Entreprise.
  3. Dans cette section, cliquez sur Modules d’affaire à la fin de la liste.
  4. Localisez le module Lightspeed, et activez le.
  5. Cliquez ensuite sur Connexion à Lightspeed.
  6. Une fenêtre vous demandant vos identifiants (adresse courriel et mot de passe) pour vous connecter sur votre compte Lightspeed apparaît alors. Si les informations que vous entrez sont valides, booxi vous présentera votre numéro de compte et les choix des boutiques et de caisse. Si ce n’est pas le cas, un message d’erreur s’affichera.
  7. Sélectionnez la boutique associée pour laquelle vous souhaitez gérer les rendez-vous.
  8. Sélectionnez la caisse préférée pour cet ordinateur, avec laquelle vous réaliserez les ventes.
  9. Sélectionnez ensuite la Catégorie de taxes et la classe de taxe par défaut qui correspondent le plus à vos taxes de travail.
  10. Cliquez sur Sauvegarder et c’est fait.

Étape 2 – Associez un service dans booxi à un article de vente dans Lightspeed

booxi vous permet, ainsi qu’à vos clients, de réserver des services. Quand vient le temps de créer une vente pour un service rendu, booxi doit savoir quel item de Lightspeed correspond à quel service. Vous pouvez associer un article de vente hors stock de Lightspeed pour chaque service de booxi en suivant les étapes suivantes :

  1. Depuis le menu principal, situé sur le côté gauche de la page, cliquez sur Services.
  2. Sélectionnez un des services de votre liste.
  3. Cliquez sur Détails du service.
  4. En haut de la section Détails du service, vous devriez voir une sélection d’articles de vente (Association avec Lightspeed). Choisissez l’article associé qui doit apparaître dans la vente lorsque ce service est donné.
  5. Cliquez sur Sauvegarder pour conserver vos changements ou sur Annuler.
  6. Répétez ces étapes pour lier tous vos services avec les différents articles de vente. Si vous ne voyez pas la sélection d’articles de vente hors stock c’est parce que votre module Lightspeed n’a pas été activé.

Étape 3 – Associez votre personnel dans booxi avec vos employés dans Lightspeed

Booxi vous permet de gérer les rendez-vous de tous les membres de votre personnel. Quand vient le temps de créer une vente pour un service rendu, booxi doit savoir quel employé a donné ce service. Vous devez associer chaque employé dans Lightspeed avec un personnel dans booxi en suivant les étapes suivantes :

  1. Dans le menu principal situé sur le côté gauche de la page, cliquez sur Personnel.
  2. Sélectionnez un des membres de votre personnel dans la liste.
  3. Cliquez sur Détails du personnel.
  4. En haut de la section Détails du personnel, vous devriez voir un menu déroulant avec le nom de vos employés (Association avec Lightspeed). Choisissez le personnel associé  dans Lightspeed qui devrait apparaître dans une vente quand ce membre de votre personnel rend un service.
  5. Cliquez sur Sauvegarder pour conserver vos changements ou sur Annuler.
  6. Répétez ces étapes pour lier tous les membres de votre personnel avec les différents employés dans Lightspeed. Si vous ne voyez pas le menu déroulant présentant vos employés c’est que votre module Lightspeed n’a pas été activé.

Étape 4 – Créez une vente dans Lightspeed depuis un rendez-vous booxi.

Maintenant que vous avez paramétré votre compte booxi pour travailler avec Lightspeed, vous pouvez créer une vente pour chaque rendez-vous.

  1. Depuis le menu principal situé à gauche de la page, cliquez sur Calendrier.
  2. Sélectionnez un rendez-vous.
  3. Appuyez sur clic-droit sur le rendez-vous directement dans le Calendrier et cliquez sur Nouvelle vente Lightspeed.
  4. Le membre de votre Personnel qui fournit ce service sera, par défaut, le même que celui qui est enregistré dans le rendez-vous sur booxi. Vous pouvez le modifier.
  5. De même, le Service est automatiquement paramétré avec les informations enregistrées dans le rendez-vous, donc avec l’article de vente si vous en avez associé un. Si ce n’est pas le cas vous pouvez sélectionner un article hors stock et le lier.
  6. Les informations du Client seront également paramétrées en fonction des détails du client enregistrés dans le rendez-vous. Si booxi ne peut pas trouver un client qui correspond à un client dans Lightspeed, vous pourrez demander à booxi de l’ajouter à votre liste de clients dans Lightspeed.
  7. Par défaut, la caisse sera celle à partir de laquelle vous créez la vente, mais vous pouvez la modifier.
  8. De même, par défaut l’employé qui crée la vente est l’utilisateur actuel de booxi. Vous avez bien sûr la possibilité de changer l’employé sélectionné (Vendu par).
  9. Cliquez sur Nouvelle Vente pour créer une vente dans Lightspeed et changer de fenêtre de navigation pour compléter votre vente, en utilisant Lightspeed.

Étape 5 – Créez un bon de travail dans Lightspeed depuis un rendez-vous booxi.

Contrairement aux ventes, Lightspeed ne fournit pas d’API pour créer un bon de travail, nous vous faisons donc sauver du temps simplement en ouvrant Lightspeed avec le client correspondant et sur la bonne page pour créer votre bon de travail. Alors même si cette option est appelée Créer un bon de travail, il n’est pas directement créé pour vous, mais cette fonction vous permet de sauver beaucoup de temps dans le processus de création.

  1. Depuis le menu principal situé à gauche de la page, cliquez sur Calendrier.
  2. Sélectionnez le rendez-vous pour lequel vous souhaitez réaliser un bon de travail.
  3. Faites un clic-droit sur le rendez-vous en question et sélectionnez l’option Nouveau bon de travail Lightspeed.
  4. La fenêtre suivante apparaît.
  • Si le client est déjà synchronisé entre Lightspeed et booxi il ne vous reste plus qu’à cliquer sur Nouveau bon de travail.
  • Si le client n’existe pas dans Lightspeed, l’option Ajouter un nouveau client dans Lightspeed apparaît. Cliquez sur Nouveau Bon de travail.

6. Complétez le bon de travail comme vous avez l’habitude de le faire dans Lightspeed.

Étape 6 – Importez vos clients Lightspeed dans booxi

Même si booxi vous permet de lier un client de booxi avec un client de Lightspeed à chaque fois que vous créez une vente ou un bon de travail,  vous pouvez souhaiter importer d’un seul coup tous vos clients de Lightspeed dans booxi et réaliser l’association en avance pour sauver du temps sur le long terme.

 

  1. Depuis le menu principal situé à gauche de la page, cliquez sur l’onglet Clients.
  2. Cliquez ensuite sur l’icône Options.
  3. Et sélectionnez Importer de Lightspeed.
  4. Cette fenêtre va apparaître. Cliquez sur le bouton Importer des clients; l’import se fera immédiatement.
  5. Et c’est terminé.

À Retenir

  • Activez et paramétrez votre module Lightspeed dans booxi.
  • Associez (liez) vos services booxi avec les articles de vente hors stock.
  • Associez (liez) votre personnel dans booxi avec vos employés dans Lightspeed.
  • Créez une vente dans Lightspeed depuis un rendez-vous booxi.
  • Créez un bon de travail dans Lightspeed depuis un rendez-vous booxi.
  • Ajoutez des nouveaux clients dans votre liste Lightspeed lorsque vous créez une vente ou un bon de travail depuis booxi.
  • Utilisez Lightspeed dans une fenêtre de navigation et booxi dans une autre, afin de pouvoir facilement naviguer entre votre ventes sur Lightspeed et vos rendez-vous sur booxi.
