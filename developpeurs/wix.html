Widget Rendez-vous
La prise de rendez-vous en ligne nécessite d’avoir un compte booxi actif. Si vous n’avez pas déjà un compte booxi, vous pouvez vous inscrire gratuitement.
Essayer gratuitement
Introduction
Vous pouvez ajouter la prise de rendez-vous en ligne à votre propre site web, simplement en intégrant le widget Rendez-vous. Vos clients pourront alors prendre rendez-vous de la même façon qu’ils le font sur votre page booxi, mais à partir de votre site web et gardant vos clients sur votre site.
Votre clée d’API booxi
La clée d’API est unique à votre entreprise et permet de connecter le widget Rendez-vous à votre compte booxi. Pour trouver votre clée d’API, connectez-vous à votre compte booxi, accédez aux détails de votre entreprise, puis défilez les détails vers le bas jusqu’au champs Clée d’API.

Instructions pour le widget
Veuillez entrer votre clée d’API dans le formulaire du générateur de code. Vous pouvez personnaliser le texte des boutons Rendez-vous (anglais et français). Vous pouvez choisir entre le mode d’affichage popup ou fenêtre flottante (par défaut). Vous pouvez aussi modifier la valeur du z-index, si par exemple la fenêtre de rendez-vous apparaît sous certains éléments de votre site web. Une fois vos choix complétés, cliquez sur le bouton Générer le code. Vous obtiendrez ainsi le code HTML requis et des instructions pour l’intégrer à votre site.
Code du widget
Veuillez entrer votre clée d’API dans le formulaire du générateur de code. Vous pouvez personnaliser le texte du bouton Rendez-vous (anglais et français). Une fois vos choix complétés, cliquez sur le bouton Générer le code. Vous obtiendrez ainsi le code HTML requis et des instructions pour l’intégrer à votre site Wix.
1. Connectez-vous à votre compte Wix et ouvrez l’éditeur de page pour la page dans laquelle vous désirez ajouter le bouton Rendez-vous.
2. Cliquez sur Add button (Ajouter un bouton) et choisissez l’option More (plus) au bas du menu Add button.
3. À partir du sous-menu Add More, choisissez l’option HTML Code située sous la section HTML & Flash.
4. Un nouvel élément a été ajouté à votre page, cliquez sur le bouton Enter Code du nouvel élément.
5. Une fenêtre de configuration identifiée HTML Settings s’affichera à l’écran. Assurez-vous que l’option Code soit sélectionnée.
6. Copiez le code ci-bas et cliquez sur la phrase "Add your code here:". Une boîte de texte s’affichera, collez le code copié dans cette boîte.
7. Cliquez sur le bouton Update. Le bouton Rendez-vous devrait apparaître à l’intérieur de l’élément.
8. Placez simplement le bouton à l’endroit désiré sur votre page web. Nous vous suggérons d’ajuster les dimenssions de l’élément pour égaler les dimenssions du bouton.
9. Testez votre site web afin de vous assurer que le bouton fonctionne correctement.
