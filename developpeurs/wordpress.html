Prise de rendez-vous pour WordPress

Widget Rendez-vous pour WordPress
Version 0.2 Beta
Requière : WP Ver 4+
Mise à jour : 2015-07-06
Download 0.2 Beta
Survol
Le widget Rendez-vous pour WordPress est une extension vous permettant d’ajouter facilement le bouton Rendez-vous de booxi à votre site web WordPress. L’utilisation du widget requiert un compte booxi et une clée d’API valide.
Instructions d’intégration
  1. Téléchargez le plugin (version 0.2 Beta)
  2. Vous devez télécharger le fichier .zip d’extension vers votre site WordPress. Pour ce faire, téléchargez l’extention à partir de cette page en utilisant le bouton Télécharger à gauche et sauvegarder l’extension sur votre ordinateur. Maintenant, ouvrez votre tableau de bord WordPress, sélectionnez le menu Extensions, puis l’option Extensions installées. À partir de votre liste d’extentions installées, sélectionnez l’option Ajouter. Cliquez sur le bouton Mettre une extension en ligne situé au haut de la liste des extentions disponible.
  3. Cliquez sur Choose file pour récupérer le fichier plugins sur votre ordinateur, et cliquez sur Installer maintenant.
  4. Maintenant retournez à vos extensions installées en sélectionnant l’option Extensions installées du menu Extensions de votre tableau de bord WordPress. Vous devez maintenant Activer l’extension widget booxi.
  5. Une fois que vous avez activé l’extention, vous obtenez un nouveau choix : booxi Widget , disponible sous le menu Réglages à partir de votre tableau de bord WordPress. Vous devez sélectionner cette option pour configurer votre bouton Rendez-vous.
  6. Vous devez entrer votre clée d’API dans l’espace Default API Key. Vous pouvez retrouver votre clée d’API dans la section Mon Entreprise, sous section Détails de votre compte booxi.
  7. Assurez-vous de choisir une langue et un libellé par défaut, lesquels seront utilisés pour afficher le bouton Rendez-vous.
  8. Votre bouton est prêt à être utilisé. Vous pouvez l’ajouter en tant que widget à votre navigation ou simplement l’ajouter à des publications en utilisant le racourci (shortcode).
Rendez-vous en tant que widget de menu
Dans le menu Apparence de votre tableau de bord WordPress et sous l’option Widgets se trouve un nouveau widget à votre disposition. A partir du menu apparence de votre tableau de bord, vous aurez un nouveau widget à votre disposition à un menu de votre thème qui prend en charge des widgets. Il ne vous suffit de glisser-déposer le widget Book Now là où vous le souhaitez. Quand un nouveau widget est créé, il a automatiquement les paramètres par défaut. Cependant, vous êtes libre de les adapter à chaque occurrence du bouton. Vous pouvez définir:
booxi Api Key: Vous pouvez entrer une clée d’API différente ici, cela vous permet d’accepter la réservation en ligne de plusieurs comptes d’affaires booxi.
Button Language: Vous pouvez spécifier le language de la fenêtre de prise de rendez-vous booxi. Nous offrons présentement une expérience de prise de rendez-vous en anglais et en français.
Button Text: est le libellé qui apparaît sur le bouton.
Ajouter le bouton Rendez-vous à des publications et pages
Il est possible d’ajouter également le bouton Rendez-vous dans une publication ou une page à l’aide d’un simple shortcode : [book-now-button]. WordPress remplacera le shortcode par un bouton Rendez-vous.

Le bouton sera généré en utilisant les paramètres par défaut, mais vous pouvez personnaliser chaque instance du bouton un peu comme pour l’utilisation du widget. Pour ce faire il ne suffit que d’ajouter l’un des attributs facultatifs suivants au shortcode:
api_key: Attribut pour utiliser une clée API différente.
lang: Attribut qui détermine la langue.
button_text: Attribut qui vous permet de personnaliser le libellé du bouton.
Exemple d’utilisation: [book-now-button api_key= "API_KEY_CODE" lang= "fre" button_text= "Rendez-vous" ].
Options de personnalisation
Plusieurs options de personnalisations sont disponibles pour vous permettre d’agencer le bouton Rendez-vous aux couleurs et style de votre site web.
Mode d’affichage du widget
Lorsque vous appuyez sur le bouton réserver, le livre Réserver un widget, sera affiché sur votre site. Il existe deux modes d’affichage disponibles:
Display in page: Le widget apparaît en avant-plan de votre site web.
Display as popup: Le widget apparaît dans une fenêtre externe par-dessus votre site web.
Paramètres d’apparence
Il est possible de personnaliser certains aspects du bouton, comme par exemples la taille et la couleur.
Button Icon: Vous pouvez choisir d’afficher ou cacher l’icône Rendez-vous affiché près du libélé.
Border Radius : Personnaliser l’arrondi des angles du bouton.
Font Size : Permet de modifier la taille du texte sur le bouton.
Button Height : Permet de modifier la hauteur du bouton.
Padding Left / Right : Permet de définir l’espacement gauche et à droite des extrêmités du bouton et du texte. La largeur du bouton s’ajuste automatiquement à la longueur du texte.
Paramètres de couleur
Vous pouvez personnaliser la couleur du texte, du fond et du contour du bouton.
Text Color : Pour la couleur du texte sur le bouton.
Background Color : Pour la couleur du bouton lui-même.
Border Color : Pour la couleur du contour de 1 pixel du bouton.
Bouton sensitif couleur
L’état de surface du bouton apparaît lorsque le curseur de la souris est au-dessus du bouton, cela sert à mettre en évidence la sélection courante. Il est possible de personnaliser les couleurs de l’état de surface.
Hover Text Color : Pour la couleur du texte sur le bouton lorsque le curseur de la souris survole le bouton.
Hover Background Color : Pour la couleur du bouton lui-même lorsque le curseur de la souris survole le bouton.
Hover Border Color : Pour la couleur du contour du bouton, lorsque le curseur de la souris survole le bouton.
CSS personnalisé
Pour les propriétaires de site web qui sont familiers avec CSS, nous avons ajouté la possibilité de définir un bloc de CSS qui sera ajouté au bouton. Vous pouvez définir un ensemble de CSS pour l’état normal et un pour l’état en surface (ou survole). Cette fonction est réservée aux utilisateurs expérimentés.
